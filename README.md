# **algorithms-and-data-structures**
Study repository for algorithms and data structures, containing exercises, implementations, notes, and learning resources.

Repositório de estudos de Algoritmos e Estruturas de Dados com exercícios, implementações e anotações.

**About**

This repository documents my studies in Algorithms and Data Structures during my Computer Science degree.

It includes:

- exercises assigned throughout the course;
- my own implementations and solutions;
- reference implementations discussed in class;
- study notes and summaries;
- experiments created to reinforce concepts.

- The goal is to organize my learning process and track my progress over time.

**Sobre**

Este repositório documenta meus estudos em Algoritmos e Estruturas de Dados durante a graduação em Ciência da Computação.

Ele reúne:

- exercícios propostos ao longo da disciplina;
- implementações desenvolvidas por mim;
- implementações de referência discutidas em aula;
- anotações e resumos de estudo;
- experimentos realizados para reforçar os conceitos.

O objetivo é organizar meu processo de aprendizagem e acompanhar minha evolução ao longo do tempo.

**AI Assistance**

Artificial Intelligence tools were used extensively throughout the development of this repository as learning assistants. They helped explain concepts, generate examples, discuss different approaches, implement algorithms, identify mistakes, and improve the documentation.

Ferramentas de Inteligência Artificial foram utilizadas de forma frequente durante o desenvolvimento deste repositório como apoio ao aprendizado. Elas auxiliaram na explicação de conceitos, geração de exemplos, discussão de diferentes abordagens, implementação de algoritmos, identificação de erros e aprimoramento da documentação.
